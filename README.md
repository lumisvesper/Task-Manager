# 🌟 TaskFlow - Gerenciador de Tarefas

Um gerenciador de tarefas simples, bonito e funcional, desenvolvido como **projeto de estudo**.

---

### ✨ Sobre o projeto

**TaskFlow** é um gerenciador de tarefas criado com o objetivo de praticar desenvolvimento, arquitetura de código e experiência do usuário.

A ideia é construir uma ferramenta leve, intuitiva e visualmente agradável para organizar tarefas do dia a dia, com foco em clareza e produtividade.

### 🚀 Funcionalidades atuais (v0.0.1)

- [x] **ColumnWheel** de horas para visualização do vencimento das tarefas
- [ ] Sistema completo de criação, edição e exclusão de tarefas
- [ ] Filtros e organização por status
- [ ] Interface responsiva e moderna

### 🛠️ Tecnologias utilizadas

- Android Studio
- Jetpack Compose
